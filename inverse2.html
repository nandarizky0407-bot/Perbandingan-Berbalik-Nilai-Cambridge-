Inverse Proportion Definition (Berbalik Nilai)

What is Inverse Proportion?

An Inverse Proportion occurs when an increase in one variable causing a proportional decrease in the other variable.

Key Characteristics:

  • If Value X goes UP (↑), Value Y goes DOWN (↓).
  • The multiplied **Product** (x × y) always hits the exact same constant (k).

The Golden Equation:

x₁ · y₁ = x₂ · y₂

To find the unknown variable, multiply the complete initial pair, then divide it by the single known variable!

Quick Summary:

Direct Proportion: Variables adjust in uniform symmetry (x/y = k). Graph is perfectly straight.

Inverse Proportion: Variables counteract symmetrically (x · y = k). Graph bends in curves.
